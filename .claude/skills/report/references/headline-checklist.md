# Headline Checklist

Every slide headline must pass these checks.

## The Checklist

For each headline, ask:

### 1. Is it a complete thought?

- [ ] Could this be a sentence (with a period at the end)?
- [ ] Does it express a finding, not just a topic?

**Bad:** "Consumer Insights"
**Good:** "Consumers Seek Mood Uplift, Not Just Indulgence"

### 2. Does it answer "so what?"

- [ ] Is there an implication or judgment, not just a description?
- [ ] Would a reader know why this matters?

**Bad:** "Concept Testing Results"
**Good:** "Hazelnut Won — But Health Claims Created Skepticism"

### 3. Is it specific enough to be memorable?

- [ ] Could this headline only apply to THIS research?
- [ ] Would it still be meaningful in 6 months?

**Bad:** "Key Findings"
**Good:** "Mini-Thin Shines in We-Time More Than Me-Time"

### 4. Does it avoid "weasel" qualifiers?

- [ ] No "may," "might," "could potentially," "seems to"
- [ ] Takes a clear position (even if conditional)

**Bad:** "Health Claims May Have Some Impact"
**Good:** "Health Claims Undermine Taste Credibility"

### 5. Is it decision-relevant?

- [ ] Does this help leadership make a decision?
- [ ] If not, does this slide belong in the deck?

**Bad:** "Respondent Demographics"
**Good:** [This slide probably shouldn't exist in the oral debrief]

## Common Transformations

| Topic Label ❌ | Finding Headline ✅ |
|----------------|---------------------|
| Concept Reactions | Hazelnut Leads — But Only When Sugar Claims Are Hidden |
| Competitive Landscape | CA! Loses to Ferrero on "Treat Yourself" — But Wins on Approachability |
| Consumer Insights | Consumers Define Premium as "Elevated Familiar," Not "Aspirational New" |
| Occasion Analysis | Work/Study Is the Key Ownable Occasion |
| Recommendations | Lead with Hazelnut, Avoid Sugar Claims, Focus on Work Moments |
| Portfolio Strategy | Premium CA! Expands the Portfolio — Cannibalization Risk Is Low |

## Quick Test

Read the headline aloud and ask: "Would a busy executive learn something useful from this?"

If no, rewrite.

## When to Break the Rules

Some slides legitimately need simple labels:
- Title slide
- Contact slide
- Methodology (but only in appendix)

The core content slides (2-13 in the decision document) should never be topic labels.
