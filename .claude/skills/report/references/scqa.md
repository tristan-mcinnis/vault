# SCQA Narrative Reference

Transform research findings into compelling strategic narratives using the SCQA framework.

## SCQA Framework

| Element | Purpose | What It Answers |
|---------|---------|-----------------|
| **Situation** | Set the context | What's the current state? What do we all agree on? |
| **Complication** | Introduce the tension | What's changed? What's the problem? Why act now? |
| **Question** | Focus the inquiry | What must we answer to resolve this? |
| **Answer** | Deliver the insight | What did we learn? What should we do? |

## Identify the Core Tension

Every good SCQA has a tension:
- Gap between expectation and reality
- Shift in the market/consumer
- Opportunity being missed
- Risk being ignored

## Element Guidelines

### Situation (1-2 paragraphs)
- Establish shared reality
- State what's known/agreed
- Set the stage without controversy
- Use data to ground the context

**Template**:
> [Client/Brand] operates in [context]. The category is characterized by [key dynamics]. Currently, [current state or approach]...

### Complication (1-2 paragraphs)
- Introduce the tension
- Show what's changed or at risk
- Create urgency
- Make the reader feel the problem

**Template**:
> However, [change/tension/problem]. This means [consequence]. The current approach [limitation]...

### Question (1 sentence)
- Crystallize the strategic question
- Make it answerable by your research
- Focus the reader

**Template**:
> The question becomes: [Strategic question]?

### Answer (Main body)
- State the core insight first
- Support with evidence
- Build the argument
- Lead to recommendations

**Template**:
> Our research reveals that [core insight]. Specifically, we found [supporting findings]...
> This suggests [strategic implications]...
> We recommend [actions]...

## Quality Guidelines

**Strong SCQA**:
- Situation is genuinely uncontroversial
- Complication creates real tension
- Question is specific and answerable
- Answer directly addresses the question
- Each element flows naturally to the next

**Weak SCQA**:
- Situation already contains the problem (too much front-loading)
- Complication is generic ("things are changing")
- Question is too broad or rhetorical
- Answer doesn't match the question
- Jumps to recommendations without insight
