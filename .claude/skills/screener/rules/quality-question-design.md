# Question Design Standards

Every screener question must meet these five standards.

## Five Principles

| Principle | Standard | Example |
|-----------|----------|---------|
| Single-barreled | Ask one thing at a time | Split "buy and wear" into two questions |
| Specific timeframes | No "recently", "often" | Use "past 3 months", "past 12 months" |
| Concrete options | Clear, comprehensive, mutually exclusive | Age ranges that don't overlap |
| Unbiased language | No leading terms | "What do you think" not "Don't you agree" |
| Numeric codes | Every option gets a code | 1, 2, 3... for data entry |

## Quick Fixes

| Bad | Good |
|-----|------|
| "How often do you buy and wear sneakers?" | Q1: Purchase frequency, Q2: Wear frequency |
| "Have you purchased recently?" | "Have you purchased in the past 3 months?" |
| "Do you like this amazing product?" | "How would you rate this product?" |

## Avoid

| Pattern | Problem |
|---------|---------|
| Double-barreled questions | Can't interpret which part the answer addresses |
| Vague timeframes | Different respondents interpret differently |
| Missing codes | Data entry errors, can't track quota |
