# China Market Localization

Standards for China-market screeners.

## Localized Retailers

Use China-specific retail channels:

| Retailer | Chinese |
|----------|---------|
| Tmall | 天猫 |
| JD.com | 京东 |
| Dewu | 得物 |
| Douyin Shopping | 抖音商城 |
| Xiaohongshu Shopping | 小红书商城 |
| WeChat Mini Programs | 微信小程序 |

## Income Thresholds (Monthly RMB)

| Range | Chinese |
|-------|---------|
| Less than ¥5,000 | 少于5,000元 |
| ¥5,000-¥9,999 | 5,000-9,999元 |
| ¥10,000-¥19,999 | 10,000-19,999元 |
| ¥20,000-¥29,999 | 20,000-29,999元 |
| ¥30,000-¥49,999 | 30,000-49,999元 |
| ¥50,000+ | 50,000元及以上 |
| Prefer not to answer | 不方便透露 |

## City Tier Classification

| Tier | Cities |
|------|--------|
| Tier 1 | Beijing, Shanghai, Guangzhou, Shenzhen |
| Tier 2 | Chengdu, Hangzhou, Wuhan, Chongqing, Nanjing, Xi'an |
| Tier 3+ | Other cities |

## Contact Collection

- **Primary:** WeChat ID (more reliable than email in China)
- **Secondary:** Mobile phone number
- **Avoid:** Email (low usage for personal communication)

## Avoid

| Pattern | Problem |
|---------|---------|
| Western retailers (Amazon, eBay) | Not relevant in China |
| USD income thresholds | Meaningless to respondents |
| Email as primary contact | Low response rate |
