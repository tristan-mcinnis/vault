# Evidence Hierarchy

Not all evidence is equal. Weight evidence by type when building findings and insights.

## Hierarchy (Strongest to Weakest)

| Rank | Type | Description | Weight |
|------|------|-------------|--------|
| 1 | **Direct observations** | Timestamped behavioral data, what people actually did | Strongest |
| 2 | **Patterns across sources** | Consistency across 3+ participants/sources | Strong |
| 3 | **Self-reported statements** | What people say they do/think/feel | Moderate |
| 4 | **Secondary research** | External data, flagged as assumptions | Weakest |

## Application

When building findings:
1. Prioritize direct observations over self-reported claims
2. Look for pattern consistency (3+ sources minimum)
3. Flag secondary research explicitly as assumptions
4. Note when self-reported contradicts observed behavior

## Say-Do Gap

Self-reported statements often differ from actual behavior. When you notice:
- **Stated preference** vs **revealed preference** (what they actually chose)
- **Recall** vs **real-time** (memory distorts)
- **Social desirability** vs **actual behavior** (people say what sounds good)

Document the gap as a finding itself.

## Confidence Mapping

| Evidence Mix | Confidence |
|--------------|------------|
| Multiple direct observations + pattern | High |
| Self-reported + pattern confirmation | Medium |
| Single source or secondary only | Low |

## Avoid

| Pattern | Problem |
|---------|---------|
| Treating all evidence equally | Unreliable findings |
| Building on secondary research alone | Assumptions not insights |
| Ignoring say-do gaps | Missing key insight |
