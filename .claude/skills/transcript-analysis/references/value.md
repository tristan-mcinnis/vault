# Value Perception Reference

Extract how consumers perceive value and worth from qualitative research data.

## Value Perception Framework

**EXPECTATION → BENEFIT → COST → WORTH JUDGMENT**
- What did they expect to get?
- What do they actually get?
- What do they pay (money, time, effort)?
- Is it "worth it"?

**VALUE IS ALWAYS RELATIVE**
- To competitors
- To past experiences
- To expectations
- To reference prices

## What to Flag

- Worth language: "worth it", "rip-off", "great value", "overpriced"
- Comparison language: "cheaper than", "better than paying for"
- ROI calculations: "cost per use", "lasts longer", "saves time"
- Trade-off discussions: "willing to pay more for", "not worth the premium"

## Value Dimensions

| Dimension | Definition | Signal Language |
|-----------|------------|-----------------|
| **Functional** | What it does for me | "Works better", "saves time" |
| **Economic** | Price-benefit calculation | "Cost per use", "pays for itself" |
| **Emotional** | How it makes me feel | "Worth it for peace of mind" |
| **Social** | How others see me | "Worth it for the status" |
| **Symbolic** | What it says about me | "Aligns with my values" |

## Value Perception Levels

| Level | Definition | Example Language |
|-------|------------|------------------|
| **High Value** | Worth more than price | "I'd pay extra", "totally worth it" |
| **Fair Value** | Price-benefit equilibrium | "Reasonable for what you get" |
| **Poor Value** | Price exceeds worth | "Not worth it", "overpriced" |
| **Uncertain** | Can't assess worth | "No idea if this is a good price" |

## Quality Benchmarks

**Weak Value Perception (avoid)**:
- "Consumers think the product is expensive and overpriced"
  - No specific price point, no benefit consideration, generic

**Strong Value Perception (aim for)**:
- "$200 feels steep initially but five-year durability justifies cost; competitors break annually requiring repurchase"
  - Specific price, initial reaction vs. justified conclusion, ROI calculation

**Weak Value Perception (avoid)**:
- "Customers feel they get good value for money"
  - Vague, no specifics about what creates value

**Strong Value Perception (aim for)**:
- "Grocery delivery's $10 fee feels expensive until comparing two-hour time save; convenience worth $5/hour calculation"
  - Specific cost, value driver identified, personal ROI calculation

## Value Perception Shifts

Many perceptions follow this pattern:
1. **Initial Judgment** (often price-based): "Too expensive"
2. **Context/Experience**: Trial, comparison, usage, time
3. **Reframed Assessment** (benefit-inclusive): "Actually worth it"

Key: Identify what triggers the shift

## Comparative Value Framework

| Comparison Type | Example | Insight |
|-----------------|---------|---------|
| **vs. Direct Competitor** | "Brand X cheaper but breaks faster" | Quality-price trade-off |
| **vs. DIY Alternative** | "$12 meal kit vs $3 groceries I won't use" | Behavioral reality check |
| **vs. Current Behavior** | "$150 gym I use vs $10 gym I don't" | Utilization-adjusted value |
| **vs. Consequence Cost** | "Organic premium vs pesticide worry" | Emotional cost-benefit |
| **vs. Time/Effort** | "$10 delivery = 2 hours saved" | Personal ROI calculation |
