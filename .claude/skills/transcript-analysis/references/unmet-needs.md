# Unmet Needs Reference

Extract gaps between what consumers want and what's currently available from qualitative research data.

## Unmet Needs Framework

**DESIRED STATE vs. CURRENT STATE = GAP**
- What do they want/need?
- What's available?
- What's missing?

**EXPLICIT → IMPLICIT → LATENT**
- Explicit: Directly stated ("I wish...")
- Implicit: Revealed through workarounds
- Latent: Problems they've adapted to

## What to Flag

- Wish statements: "I wish there was...", "If only...", "Why doesn't anyone make..."
- Workarounds: "I have to... instead", "I hack together..."
- Compromises: "I settle for...", "It's good enough but..."
- Frustrations: "The problem is...", "I spend hours trying to..."

## Unmet Need Categories

| Category | Definition | Signal Language |
|----------|------------|-----------------|
| **Functional** | Performance gaps | "It should do X but doesn't" |
| **Experiential** | Journey gaps | "The process should be easier" |
| **Emotional** | Feeling gaps | "I want to feel X but feel Y" |
| **Social** | Relationship gaps | "I need help/community/recognition" |
| **Integration** | System gaps | "These should work together" |

## Quality Benchmarks

**Weak Need (avoid)**:
- "Consumers need better quality products"
  - Vague, not actionable, no specific gap

**Strong Need (aim for)**:
- "Need activewear sizing between standard and plus; current gap forces uncomfortable fit compromises"
  - Specific gap, clear problem, actionable

**Weak Need (avoid)**:
- "Consumers want fast delivery"
  - Solution exists widely, not truly unmet

**Strong Need (aim for)**:
- "Need same-day delivery for urgent baby supplies without subscription commitment; ad-hoc emergency access unavailable"
  - Specific context, current limitation, desired solution

## Workaround Analysis

Workarounds = Evidence of unmet needs

| Workaround | Underlying Need |
|------------|-----------------|
| Using multiple apps together | Integration need |
| Modifying products after purchase | Customization need |
| Creating spreadsheets/checklists | Organization need |
| Asking friends for help | Expert guidance need |
