# Trends & Future Expectations Reference

Extract emerging patterns and consumer predictions about market evolution from qualitative research data.

## Trends Analysis Framework

**WHAT'S CHANGING + WHY + IMPLICATIONS**
- What behavior/attitude is shifting?
- What's driving the change?
- What does it mean for markets?

**TRENDS vs. EXPECTATIONS**
- Trends: Change already underway ("I used to... now I...")
- Expectations: Desired/predicted change ("I expect...", "Brands should...")

## What to Flag

- Change language: "I used to... but now...", "People are starting to..."
- Future orientation: "I expect...", "In the future...", "Brands should..."
- Value shifts: "I don't care about... anymore", "...matters more now"
- Disruption signals: "The whole model is...", "Why do we still..."

## Trend Types

| Type | Definition | Example Signal |
|------|------------|----------------|
| **Behavioral** | How people do things differently | "I used to shop in-store; now I..." |
| **Attitudinal** | Belief/value changes | "Quality matters less than..." |
| **Expectation Inflation** | Rising standards | "That's table stakes now" |
| **Category Disruption** | Fundamental rethinking | "The whole model needs to change" |
| **Generational** | Younger cohorts' different values | "My kids would never..." |

## Future Expectation Types

| Type | Definition | Example |
|------|------------|---------|
| **Evolutionary** | Incremental improvements | "It should be faster/easier" |
| **Revolutionary** | Fundamental transformation | "The whole experience needs reimagining" |
| **Normative** | "Should" statements | "Brands should take responsibility for..." |
| **Aspirational** | Desired but uncertain | "I hope someday we'll..." |

## Quality Benchmarks

**Weak Trend (avoid)**:
- "Consumers increasingly care about sustainability"
  - Widely known, not specific, not actionable

**Strong Trend (aim for)**:
- "Sustainability fatigue drives demand for pre-vetted brands; consumers want defaults not decisions"
  - Counter-trend, psychological insight, specific opportunity

**Weak Expectation (avoid)**:
- "People expect technology to continue improving"
  - Generic, obvious, no direction

**Strong Expectation (aim for)**:
- "Consumers expect AI assistants to anticipate needs proactively; reactive help feels outdated"
  - Specific technology, clear shift, behavioral implication
