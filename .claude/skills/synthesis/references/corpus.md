# Corpus Synthesis Reference

Synthesize findings across multiple research sources to surface overarching themes and patterns.

## Gathering Sources

Identify all sources for synthesis:
- **Primary**: Transcripts, insight extractions, field notes
- **Secondary**: Articles, reports, competitor analysis
- **Prior work**: Previous project learnings

## Source Inventory

Create source inventory table:

| Source | Type | Date | Key Focus |
|--------|------|------|-----------|
| MG1 Shanghai | Primary - Focus Group | Feb 5 | Premium perception |
| MG2 Guangzhou | Primary - Focus Group | Feb 7 | Regional differences |
| Market Report | Secondary | Jan 2026 | Category trends |

## Theme Extraction

Across all sources, identify:

1. **Convergent themes**: Patterns appearing across multiple sources
2. **Divergent themes**: Contradictions or tensions between sources
3. **Emergent themes**: Unexpected patterns not in research questions
4. **Contextual factors**: Variables that explain differences

## For Each Theme

- **Name the theme**: Descriptive 3-5 word title
- **Write the synthesis**: 2-3 sentence summary of the pattern
- **Evidence breadth**: How many sources support this?
- **Key quotes**: Representative verbatims from each source
- **Nuances**: Important caveats or variations

## Quality Guidelines

**Good Synthesis**:
- Elevates beyond individual sources
- Names patterns that weren't explicit in any single source
- Acknowledges tensions rather than forcing false consistency
- Identifies surprising emergent findings

**Bad Synthesis**:
- Just lists findings from each source
- Ignores contradictions
- Only confirms what was expected
- Loses the richness of individual sources
