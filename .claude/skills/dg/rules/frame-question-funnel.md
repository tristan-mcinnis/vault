# Question Flow Design (Funnel Approach)

Design questions in depth levels, progressing from concrete behaviors to strategic implications.

## Depth Levels

| Level | Focus                 | Type | Example |
| ----- | --------------------- | ---- | ------- |
| L1    | What happened?        | Behavioral | "Walk me through a typical shopping trip" |
| L2    | Why that way?         | Motivational | "What made you choose that store?" |
| L3    | What does it mean?    | Meaning | "What does that choice say about you?" |
| L4    | What else is true?    | Connections | "How does that connect to other choices?" |
| L5    | Strategic implication | Projection | "If [brand] did X, how would you react?" |

## Application Principles

### Section Flow
Each section should progress through at least 3 depth levels:
- **Open with L1-L2** ‚Äî Establish behavioral facts and surface motivations
- **Build to L3** ‚Äî Explore meaning and emotional significance
- **Close with L4-L5** ‚Äî Connect to broader patterns or test strategic directions

### Question Sequencing
- Start broad, then narrow
- Use L1-L2 to build rapport and establish shared language
- Reserve L5 (projection) questions for:
  - End of sections
  - Concept evaluation
  - Future visioning exercises

### Timing by Level

| Level | Typical Time | Notes |
| ----- | ------------ | ----- |
| L1 | 1-2 min | Quick recall, grounding |
| L2 | 2-3 min | May need probes |
| L3-L4 | 3-4 min | Requires reflection |
| L5 | 2-4 min | Depends on complexity |

## Common Mistakes

- **Front-loading L3-L5** ‚Äî Asking "what does it mean?" before establishing "what happened?"
- **Staying at L1-L2** ‚Äî Gathering facts without exploring significance
- **Jumping levels** ‚Äî Going from L1 directly to L5 without building understanding
- **Overusing L5** ‚Äî Projection fatigue if every question asks "what if?"
