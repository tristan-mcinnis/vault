# Timing Constraints

Guidelines for realistic timing allocation in discussion guides.

## Session Duration Mapping

| Session Type | Duration | Sections | Avg Section Time |
| ------------ | -------- | -------- | ---------------- |
| 90-min FGD | 90 min | 4-5 | 18-22 min |
| 2-hr FGD | 120 min | 5-6 | 20-24 min |
| 2.5-hr FGD | 150 min | 6-7 | 21-25 min |
| 60-min IDI | 60 min | 5-6 | 10-12 min |
| 90-min IDI | 90 min | 5-6 | 15-18 min |

## Section Time Budgeting

### Standard Sections
| Section Type | Recommended Time |
| ------------ | ---------------- |
| Warm-up/Introduction | 5-10 min |
| Context/Occasion | 15-20 min |
| Brand/Competition | 20-25 min |
| Concept Evaluation (per concept) | 4-5 min (IDI) / 7-8 min (FGD) |
| Wrap-up/Close | 5-10 min |

### Built-in Buffers
Always account for:
- **Transitions**: 1-2 min between sections
- **Stimulus handling**: 1-2 min for distribution/display
- **Natural tangents**: 5-10% of total time
- **Late starts**: 5 min buffer at beginning

## Validation Rules

### Time Sum Check
```
Section time = Sum of part times
Total time = Sum of section times + transitions
Total time ≤ Session duration
```

### Questions-to-Time Ratio
| Section Duration | Max Questions |
| ---------------- | ------------- |
| 10 min | 3-4 |
| 15 min | 5-6 |
| 20 min | 6-8 |
| 25 min | 8-10 |
| 30 min | 10-12 |

### Red Flags
- Section time doesn't match sum of parts
- Total exceeds session duration by >10%
- 10+ questions in sections under 25 min
- Concept evaluation budgeted at <4 min each

## Concept Evaluation Timing

### Time Per Concept
| Method | Time per Concept | Questions per Concept |
| ------ | ---------------- | --------------------- |
| IDI | 4-5 min | 6-9 (mix of quick-fire and depth) |
| FGD | 7-8 min | 4-6 + individual write |

### Question-Type Timing Within Concepts
| Question Type | Time |
| ------------- | ---- |
| First reaction (Q1-2) | ~30 sec |
| Purchase intent (yes/no) | ~30 sec |
| Believability probe | 1-2 min |
| Brand fit probe | 1-2 min |
| Competitive comparison | 1-2 min |

**Key insight:** Not all concept questions are equal duration. Budget 30 sec for quick-fire, 1-2 min for depth probes.

### Maximum Concepts
- **IDI**: 4-6 concepts max
- **FGD**: 4-6 concepts max (consider rotation)

## Page Break Conventions

| Element | Page Break |
| ------- | ---------- |
| New section | Always starts new page |
| New part | Same page unless >3/4 down |
| Stimulus material | Own page or clearly demarcated |
| Exercises | Own page if instructions needed |
