# Question Writing Principles

Standards for writing effective discussion guide questions.

## Core Principles

| Principle | Standard |
| --------- | -------- |
| **Numbering** | All questions must be sequentially numbered within each section (Q1, Q2, Q3...) |
| **Simplicity** | Single-barreled, stand-alone questions |
| **Questions per section** | 4-9 questions |
| **Time per question** | 2.5-4 minutes average |
| **Follow-ups** | Combine related probes naturally |
| **Clarity** | Avoid "optional probe" structures (must-ask unclear) |

## Single-Barreled Questions

**Good:**
- "What made you choose this brand?"
- "How do you typically discover new products in this category?"

**Bad (multi-barreled):**
- "What made you choose this brand and how does it compare to what you used before?"
- "Tell me about your shopping habits and how they've changed recently."

Split compound questions into separate questions with clear sequencing.

## Section Density

| Questions per Section | Assessment |
| --------------------- | ---------- |
| 1-3 | Too sparse — consider combining sections |
| 4-9 | Optimal range |
| 10+ | Too dense — split section or prioritize |

### Calculation
- 25-min section ÷ 2.5 min/question = ~10 questions max
- But allow buffer for probing and natural discussion
- Target 6-8 questions for a 25-min section

## Timing Realism

### By Question Type
| Type | Time |
| ---- | ---- |
| Quick-fire (yes/no, rating) | 30-45 sec |
| Factual recall (L1) | 1-2 min |
| Motivational (L2) | 2-3 min |
| Meaning/depth (L3-L4) | 3-4 min |
| Projection (L5) | 2-4 min |

### Red Flags
- 10+ questions in a 25-min section
- No buffer time for natural discussion
- Concept evaluation budgeted at 2 min each (typically needs 4-5)

## Probe Structure

**Clear hierarchy:**
```
Q1. Main question here?
    - Probe: Follow-up for depth
    - Probe: Alternative angle
    [Moderator note: Listen for specific cues]
```

**Avoid:**
- "Optional probe:" (ambiguous priority)
- Probes that are actually new questions
- More probes than the main question can support

## Open vs. Closed Questions

**Open (preferred for depth):**
- "Tell me about..."
- "How do you..."
- "What comes to mind when..."

**Closed (use sparingly):**
- Screening/verification: "Have you purchased this in the last month?"
- Rating capture: "On a scale of 1-5..."
- Quick-fire reactions: "First word that comes to mind?"

Closed questions are tools for specific purposes, not default question style.
