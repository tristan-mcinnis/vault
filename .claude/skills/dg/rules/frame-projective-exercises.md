# Projective Exercises

Include 1-2 projective exercises per DG based on research objectives. These techniques help respondents articulate abstract concepts and emotional associations.

## Exercise Types

| Exercise | When to Use | Format |
| -------- | ----------- | ------ |
| **Mapping** | Category/brand positioning | Place brands on 2x2 matrix or spectrum |
| **Personification** | Brand/product personality | "If this brand were a person..." |
| **Collaging** | Emotional territory | Select images representing feelings |
| **Future visioning** | Innovation, aspiration | "Imagine 5 years from now..." |
| **Cultural codes** | Category meaning, symbolism | Explore cultural associations |

## Exercise Selection Guide

### Mapping
**Best for:**
- Competitive landscape understanding
- Price-quality perception
- Occasion-brand fit

**Format options:**
- 2x2 matrix with client-defined axes
- Linear spectrum (e.g., traditional → modern)
- Free-form territory mapping

**Timing:** 5-8 min (individual placement + discussion)

### Personification
**Best for:**
- Brand personality research
- Emotional brand associations
- Competitive differentiation

**Common prompts:**
- "If [brand] walked into a party, what would they be like?"
- "What celebrity would this brand be?"
- "What car would this brand drive?"

**Timing:** 5-7 min per brand

### Collaging
**Best for:**
- Emotional territory definition
- Visual language development
- Abstract concept exploration

**Format:**
- Pre-selected image sets (8-12 images)
- Free selection from magazines/digital library
- Mood board creation

**Timing:** 10-15 min (selection + explanation)

### Future Visioning
**Best for:**
- Innovation ideation
- Unmet needs exploration
- Aspiration mapping

**Common prompts:**
- "Imagine the perfect [product] 5 years from now"
- "What would make you upgrade?"
- "How would your ideal [experience] differ?"

**Timing:** 5-8 min

### Cultural Codes
**Best for:**
- Cross-cultural research
- Category meaning exploration
- Symbol/ritual identification

**Format:**
- Cultural associations mapping
- Ritual/routine walkthrough
- Symbol decoding exercise

**Timing:** 8-12 min

## Integration Principles

1. **Position strategically** — Place after context-building, before evaluation
2. **Warm up first** — Don't start cold; ensure respondents are comfortable
3. **Use for triangulation** — Complement verbal responses with projective insights
4. **Adapt for method** — IDI allows deeper individual exploration; FGD benefits from group energy

## Anti-Patterns

- Using 3+ projective exercises (exercise fatigue)
- Placing projectives at session start (no rapport)
- Using mapping without clear axis definitions
- Expecting projectives to replace direct questioning
