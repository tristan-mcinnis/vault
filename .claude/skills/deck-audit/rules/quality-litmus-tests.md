# Quality Litmus Tests

Quick checks to assess deck quality. Run these before diving into detailed analysis—they often reveal fundamental issues.

## The Five Tests

### 1. CEO Test

**Question:** Could a time-pressed executive get the point from titles + exec summary alone?

**Pass:** Reading just slide titles tells a coherent story; exec summary captures the insight.

**Fail:** Titles are topic labels ("Market Overview"); exec summary is vague or missing.

**Why it matters:** Most senior audiences skim. If the deck doesn't work at skim depth, it fails.

---

### 2. Competitor Test

**Question:** Would you be worried if a competitor saw this deck?

**Pass:** Deck contains proprietary insights, specific recommendations, competitive intelligence.

**Fail:** Everything in the deck is public knowledge or obvious.

**Why it matters:** If a competitor wouldn't care, the insight may be too weak to matter.

---

### 3. Delete Test

**Question:** If you removed slides 6-10 (or any middle section), would anyone notice the gap?

**Pass:** Removing any section creates a logical hole in the argument.

**Fail:** Slides could be cut and the story still flows.

**Why it matters:** Every slide should earn its place. Filler dilutes impact.

---

### 4. 5-Year Test

**Question:** Will this insight still feel true in 5 years, or is it merely timely?

**Pass:** Core insight reflects a durable truth about the market or consumers.

**Fail:** Insight is just "things are changing" or relies on temporary conditions.

**Why it matters:** Timeless insights command premium; reactive observations are commodity.

---

### 5. Reverse Test

**Question:** Could you argue the opposite position with equal conviction?

**Pass:** The opposite view would be clearly wrong given the evidence.

**Fail:** "Consumers want quality" — well, what's the alternative? "Consumers want garbage"?

**Why it matters:** If both sides of the claim are equally defensible, the insight is too obvious.

## Using These Tests

- Run all 5 before detailed review (~5 minutes)
- Two or more failures suggest fundamental issues
- Document which tests pass/fail in the audit summary
- A deck can pass detailed Macro/Meso/Micro analysis but still fail litmus tests

## Avoid

| Pattern | Problem |
|---------|---------|
| Skipping tests for time | These are fast; fundamental issues are expensive to miss |
| Over-weighting one test | All five together give a balanced quick read |
| Applying to wrong deck type | Status updates don't need to pass the Competitor Test |
