# Calibration by Deck Type

Different presentations have different standards. Calibrate expectations before auditing.

## Deck Type Standards

| Deck Type | Key Emphasis | Acceptable Trade-offs | Typical Length |
|-----------|--------------|----------------------|----------------|
| **Strategy recommendation** | SCQA structure, clear recs, logical flow | Can be longer, more evidence-heavy | 20-40 slides |
| **Research debrief** | Evidence quality, insight depth, quote attribution | May have more data slides, less polish | 30-50 slides |
| **Pitch deck** | Narrative punch, memorability, emotional hook | Can be lighter on evidence | 10-15 slides |
| **Status update** | Clarity, brevity, actionable next steps | Lower bar on insight depth | 5-15 slides |
| **Workshop materials** | Usability, discussion prompts, participant engagement | Can be more fragmented, less narrative | Varies |
| **Board presentation** | Executive clarity, risk framing, decision focus | Brevity over completeness | 10-20 slides |

## Adjusting Standards

### Strategy Recommendation
- **Must have:** Clear SCQA arc, prioritized recommendations, evidence chain
- **Can tolerate:** Longer length, dense data slides, technical appendix
- **Watch for:** Recommendation pile-up, missing "so what" synthesis

### Research Debrief
- **Must have:** Rigorous evidence, verbatim quotes with attribution, clear methodology
- **Can tolerate:** Less polish, more data-heavy, longer format
- **Watch for:** Missing synthesis, orphan data, quote walls without insight

### Pitch Deck
- **Must have:** Memorable hook, clear ask, emotional resonance
- **Can tolerate:** Lighter evidence, assertion over proof, brevity
- **Watch for:** Unsubstantiated claims, missing credibility signals

### Status Update
- **Must have:** Clear progress indicators, blockers identified, next steps
- **Can tolerate:** Less narrative flow, simpler visuals, template use
- **Watch for:** Filler slides, unclear accountability, buried bad news

### Workshop Materials
- **Must have:** Clear exercises, participant instructions, discussion prompts
- **Can tolerate:** Fragmented flow, placeholder content, less polish
- **Watch for:** Unclear instructions, missing timing, no facilitation notes

## Scoring Implications

Adjust grade interpretation by deck type:

| Deck Type | "Good" Score | Notes |
|-----------|--------------|-------|
| Strategy recommendation | 85+ | High bar; represents the firm |
| Research debrief | 80+ | Evidence quality matters more than polish |
| Pitch deck | 80+ | Memorability can offset minor issues |
| Status update | 75+ | Functional clarity is primary |
| Workshop materials | 70+ | Usability trumps aesthetics |

## Avoid

| Pattern | Problem |
|---------|---------|
| Same standard for all decks | A workshop deck isn't a strategy deck |
| Excusing weak thinking | "It's just a status update" doesn't excuse circular logic |
| Over-calibrating | Don't drop standards so far that nothing fails |
