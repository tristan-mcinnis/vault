# Proposal Type Adaptations

Different project types require different emphasis in proposal structure.

## Type Matrix

| Type | Emphasis | Typical Length | Key Sections |
|------|----------|----------------|--------------|
| **Consumer research** | Sample design depth, methodology rationale | 15-25 pages | Detailed profiles, screening criteria, phase outputs |
| **Brand/positioning** | Thought-starters, semiotics rationale | 20-30 pages | Strong POV section, cultural context, strategic questions |
| **Retail/experience** | Journey mapping, scope definition | 15-25 pages | Touchpoint analysis, experience phases, observation methods |
| **Foresight/trends** | Expert sources, cultural analysis | 15-20 pages | Signal scanning approach, expert network, horizon framework |

## Adaptation Guidelines

### Consumer Research
- Lead with "who we're talking to and why"
- Sample design section is the star
- Methodology shows rigor and recruitment feasibility
- Include screener logic preview

### Brand/Positioning
- Lead with cultural/market context
- Thought-starters demonstrate IC's strategic POV
- Include semiotics or cultural analysis rationale
- Strategic questions frame the inquiry

### Retail/Experience
- Lead with journey or experience map concept
- Define scope boundaries clearly (which touchpoints, channels)
- Include observation/ethnographic components
- Show how insights connect to experience design

### Foresight/Trends
- Lead with the change/tension being explored
- Show expert network or signal sources
- Define time horizons (near/mid/far)
- Include scenario or implications framework

## Avoid

| Pattern | Problem |
|---------|---------|
| Same structure for all types | Misses what matters to this client |
| Consumer research depth on foresight project | Overwhelms with irrelevant sample detail |
| Light thought-starters on brand project | Fails to show IC's strategic value |
