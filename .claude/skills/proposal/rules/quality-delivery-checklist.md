# Delivery Checklist

Validate all items before sending proposal to client.

## Required Checks

| Area | Check | Critical? |
|------|-------|-----------|
| **Cover** | Correct client name | Yes |
| **Cover** | Correct project name | Yes |
| **Cover** | Current date | Yes |
| **Cover** | "Private & Confidential" present | Yes |
| **Navigation** | Page/slide numbers present | No |
| **Content** | No placeholder text remaining | Yes |
| **Content** | Tables properly formatted | No |
| **Investment** | Line items sum to totals | Yes |
| **Investment** | Currency symbols consistent | No |
| **Contact** | Information is current | Yes |
| **File** | Saved to correct location | No |
| **File** | Follows naming convention | No |

## Common Issues

| Issue | Prevention |
|-------|------------|
| Wrong client name from template | Search document for previous client names |
| Stale dates | Check cover, headers, timeline |
| Math errors in investment | Verify subtotals and grand total |
| Missing confidentiality | Check cover page and headers |

## Avoid

| Pattern | Problem |
|---------|---------|
| Skipping checklist for "quick" revisions | Small changes can introduce errors |
| Checking only changed sections | Template content may have stale data |
| Trusting auto-calculated totals | Formulas can break in copy/paste |
