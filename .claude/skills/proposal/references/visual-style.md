# Visual Style Guide

Inner Chapter proposal visual standards.

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Navy | #0A1F44 | Section dividers, borders, emphasis |
| Black | #000000 | Primary text, titles |
| Dark Gray | #262626 | Body text, secondary content |
| Light Gray | #F2F2F2 | Table header backgrounds |
| White | #FFFFFF | Backgrounds, negative space |

## Typography

### PowerPoint

| Element | Font | Size | Weight | Color |
|---------|------|------|--------|-------|
| Title slide | Aptos / Calibri | 28pt | Bold | Navy |
| Section header | Aptos / Calibri | 24pt | Bold | Black |
| Slide title | Aptos / Calibri | 20pt | Bold | Black |
| Body text | Aptos / Calibri | 14pt | Regular | Dark Gray |
| Table content | Aptos / Calibri | 12pt | Regular | Black |
| Footnotes | Aptos / Calibri | 10pt | Regular | Dark Gray |

### Word

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Document title | Aptos / Calibri | 16pt | Bold |
| H1 (Section) | Aptos / Calibri | 14pt | Bold |
| H2 (Subsection) | Aptos / Calibri | 12pt | Bold |
| H3 | Aptos / Calibri | 11pt | Bold |

## Tables

### Standard table format

```
| Header row background: #F2F2F2 |
| Border: 1pt solid #CCCCCC |
| Header text: Bold |
| Cell padding: 6px |
```

### Investment table format

- Header row: Gray background (#F2F2F2)
- Subtotals: Italic
- Totals: Bold
- Right-align currency columns
- Include currency symbol (¥, $)

## Slide Layouts

### Title Slide
```
[Logo top-left corner if applicable]

                    PROJECT TITLE
                    Subtitle / Client Name

                    Inner Chapter
                    Date

[Private & Confidential - bottom center]
```

### Section Divider
```
[Navy left border or full navy background]

        01. SECTION NAME
```

### Content Slide
```
SLIDE TITLE

[Content area - one main idea]
[Supporting visual, table, or bullets]

[Page number - bottom right]
```

### Two-Column
```
SLIDE TITLE

[Left column]          [Right column]
Content or visual      Content or visual
```

## Visual Principles

### Do
- One idea per slide
- Use tables over long bullet lists
- Left-align body text
- Use white space generously
- Keep charts simple and labeled

### Don't
- Crowd slides with text
- Use more than 3 levels of bullets
- Center large blocks of text
- Use decorative fonts
- Include unlabeled visuals

## Thought-starter Slides

Each thought-starter gets 1-2 slides:

**Slide 1: Setup**
```
THOUGHT-STARTER N: [TITLE]

[Bold observation statement]

[Evidence paragraph or table]
```

**Slide 2: Strategic Question**
```
[Continuation of evidence if needed]

        "Strategic Question: How might we [...]?"
```

## Team Slides

### Grid format (4-6 people)
```
TEAM

[Photo] Name          [Photo] Name
Title                 Title
1-line expertise      1-line expertise

[Photo] Name          [Photo] Name
...                   ...
```

### Individual format (key people)
```
NAME — TITLE

[Photo left side]

[2-3 paragraphs of background and relevant experience]

[Past work bullet list]
```
