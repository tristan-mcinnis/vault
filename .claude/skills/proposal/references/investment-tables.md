# Investment Tables

Standard formats for proposal pricing.

## Basic Format

```markdown
## Investment

| Phase | Activity | Cost |
|-------|----------|------|
| **Phase 1: Foundation** | | |
| | Desk research & competitive mapping | $X |
| | Expert interviews (x3) | $X |
| | Stimulus development | $X |
| | *Subtotal* | *$X* |
| **Phase 2: Consumer Research** | | |
| | Project management & recruitment | $X |
| | Fieldwork: IDIs (x8) | $X |
| | Incentives | $X |
| | *Subtotal* | *$X* |
| **Phase 3: Synthesis** | | |
| | Analysis | $X |
| | Reporting | $X |
| | *Subtotal* | *$X* |
| **Total (before tax)** | | **$X** |
| **Total (incl. 6% VAT)** | | **$X** |
```

## Detailed Breakdown Format

For larger projects, show line-item detail:

```markdown
## Investment Breakdown

### Phase 1: Foundation
| Item | Cost (CNY) |
|------|------------|
| Brand asset & report review | 21,200 |
| Semiotic analysis (competitive) | 42,400 |
| **Subtotal** | **63,600** |

### Phase 2: Consumer Immersion
| Item | Cost (CNY) |
|------|------------|
| Project management | 10,600 |
| Recruitment & incentives | 16,960 |
| Moderation (6 x 2hr) | 50,880 |
| Topline report | 30,000 |
| **Subtotal** | **108,440** |

### Phase 3: Synthesis
| Item | Cost (CNY) |
|------|------------|
| Full analysis | 52,250 |
| Final reporting | 55,000 |
| **Subtotal** | **107,250** |

---

**Total (before 6% VAT):** ¥279,290
**Total (after 6% VAT):** ¥296,047
```

## Multi-Market Format

```markdown
## Investment

| Market | Components | Cost |
|--------|------------|------|
| **Analysis** | Global synthesis | $28,000 |
| **NYC** | 2 x Group discussions | $20,000 |
| **Tokyo** | 6 x In-home visits | $30,000 |
| | 2 x Expert interviews | $5,000 |
| | *Subtotal* | *$35,000* |
| **Shanghai** | 2 x Group discussions | $18,000 |
| | 2 x Expert interviews | $6,000 |
| | *Subtotal* | *$24,000* |
| **Seoul** | 2 x Expert interviews | $6,600 (waived) |

**Total: $110,000 USD (incl. VAT)**
```

## Inclusions/Exclusions

Always state what's included and excluded:

```markdown
**Includes:**
- Recruitment and incentives
- Recording and transcription
- Focus group/interview venue (Inner Chapter office)
- 2 rounds of reporting revision

**Excludes:**
- Simultaneous interpretation
- External venue rental
- Travel outside Shanghai (quoted separately if needed)
- Quantitative validation
```

## Optional/Value-Add Items

When offering extras:

```markdown
### Additional Outputs (Optional)

| Output | Description | Cost |
|--------|-------------|------|
| Podcast summary | 8-10 min audio synthesis | Included (value-add) |
| Planning sessions | 2hr workshop per market | $5,000/market |
| Video highlights | 3-min stakeholder video | $8,000 |
```

## Discount Presentation

When offering discounts:

```markdown
| | Cost |
|---|------|
| Original total (incl. 6% VAT) | ¥512,722 |
| **Discounted total (incl. 6% VAT)** | **¥460,000** |

*Discount reflects: existing client relationship, strategic priority project*
```

## Currency Notes

- CNY/RMB for China-based projects
- USD for global/multi-market projects
- Always specify tax treatment (VAT included/excluded)
- Note payment terms if relevant (50% upfront, 50% on delivery typical)
